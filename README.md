# McCoy Software Tools Website

A static website built for GitHub Pages.

## Publish with GitHub Pages

1. Create a new public GitHub repository.
2. Upload `index.html`, `styles.css`, and `script.js` to the repository root.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose `main` and `/ (root)`, then save.
6. GitHub will publish the site at your GitHub Pages URL.

## Custom domain

After purchasing a domain, add it under **Settings → Pages → Custom domain**, then configure the DNS records with your registrar as GitHub instructs.
